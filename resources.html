Other resources

Swiss Year of Scientometrics (SYoS):

  • SYoS Blogpost: SYoS is another project led by our team to promote responsible research assessment in Switzerland

Open scholarly metadata databases:

  • BASE (Bielefeld Academic Search Engine)
  • OpenAIRE
  • OpenAlex
  • OpenCitations
  • Semantic Scholar

Dashboards based on open scholarly metadata databases:

  • Swiss Open Access Monitor: An Open Access Monitor for Switzerland with a Journal Monitor based on OpenAlex as well as other data sources, and a Repository Monitor based on data provided by participating institutions
  • COKI Open Access Dashboard: An Open Access Dashboard for countries and institutions worldwide based on OpenAlex, Crossref Metadata, Unpaywall and ROR
  • National Open Access Monitor Ireland: An Open Access Monitor for Ireland based on OpenAIRE
  • Hybrid Open Access Dashboard (HOAD): A dashboard highlighting the extent to which hybrid journals are part of transformative agreements, based on Crossref, OpenAlex and other sources

Knowledge maps based on open scholarly metadata databases:

  • Open Knowledge Maps: A knowledge map based on PubMed or BASE
  • SemOpenAlex Explorer: A knowledge graph based on OpenAlex

Further resources:

  • Curtis Open Knowledge Initiative - Open Metadata Report: A report comparing open bibliometric databases at a global scale
  • Data Quality in Scientometric Databases Datasets: Bibliography and reading notes on Data Quality and Difference in Bibliometric Databases (until 2021)
  • VOSviewer: A software tool for constructing and visualizing bibliometric networks based on a selected source including OpenAlex, Crossref and Europe PMC
